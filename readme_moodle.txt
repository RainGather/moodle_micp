mod_micp
========

Component: mod_micp
Plugin type: Activity module
Requires: Moodle 5.0, PHP 8.1+

Third-party libraries
---------------------
This plugin does not currently bundle any third-party libraries.

Release packaging
-----------------
The repository root is the plugin root. Release archives should unpack to:

    mod/micp

Repository-only examples are stored in the examples/ directory and are excluded
from release archives through .gitattributes.

Documentation
-------------
General plugin documentation is kept in:
* README.md
* INSTALL.md
* CHANGES.md
* SECURITY.md
